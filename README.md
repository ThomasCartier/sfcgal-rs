# sfcgal-rs
## *(WIP)*

Rust bindings to [`SFCGAL`](http://oslandia.github.io/SFCGAL/) C API.  
Based on the [sfcgal-sys](https://github.com/mthh/sfcgal-rs) crate exposing low-level bindings.
